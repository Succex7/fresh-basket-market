# FreshBasket Landing Page

A clean, modern, and fully responsive grocery delivery landing page built using **HTML and CSS only**.  
This project focuses on layout structure, responsiveness, and visual hierarchy, and is deployed using **GitHub Pages**.

## Features
- Fully responsive design (mobile, tablet, desktop)
- Clean and modern UI
- Semantic HTML5 structure
- CSS Flexbox layout
- Smooth and readable content flow

## Built With
- HTML5  
- CSS3 (Flexbox)  
- GitHub Pages
